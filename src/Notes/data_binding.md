

# Data-Binding
    ->Data Binding allows data to flow between TypeScript and HTML in an Angular application.

    ->Displaying the name variable from app.ts in app.html using {{ name }} is called Interpolation, which is a type of One-Way Data Binding in Angular.

    app.ts  ─────▶  app.html

    ->TypeScript ka data HTML me aa raha hai = Data Binding

    ----------------------------------------------------------------------------------------------------------------
    
# Types of Data-Binding
    Interpolation {{ }} → TS → HTML

    Property Binding [ ] → TS → HTML

    Event Binding ( ) → HTML → TS

    Two-Way Binding [( )] → TS ↔ HTML
